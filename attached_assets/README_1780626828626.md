# AGS Live Hall of Legends React Demo

Drop this into your Replit React app.

## File locations

Put your background here:

public/assets/scenes/scene_hall_of_legends_stage_lit.png

Put your transparent guitar PNG here:

public/assets/guitars/guitar_frankenbolt.png

## Use

Import the component:

import HallOfLegendsLive from "./components/HallOfLegendsLive";

Then render:

<HallOfLegendsLive />

## What it does

- Bright stage-lit Hall background
- Spinning vortex clockwise
- Guitar spinning the opposite direction
- Floating/bobbing guitar
- Rising energy particles
- Sweeping stage lights
- Reward panel
